Persian Parade NYC Website - Rebuilt in React

Project website: https://persianparade.com/

Description

This repository contains the source code for the Persian Parade NYC website, rebuilt in React.

Languages

- JavaScript (~94%)
- CSS
- Dockerfile
- HTML

Prerequisites

- Node.js (v14 or newer recommended)
- npm

Getting started

1. Clone the repository:

   git clone https://github.com/ColbyHman/persian-parade-website.git
   cd persian-parade-website

2. Install dependencies:

   npm install

3. Start the development server:

   npm run start

Development workflow

- Create a topic branch for your work:

  git checkout -b feature/my-change

- Make your changes, commit, and push the branch.

- Open a Pull Request against the main branch. All changes must be made via Pull Request; direct pushes to main are discouraged.

Contributing

We welcome contributions. Please open a Pull Request with a clear description of your changes and include any relevant screenshots or testing notes.
