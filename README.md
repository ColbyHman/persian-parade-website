# Persian Parade NYC Website (React Rebuild)
This repository contains the source code for the Persian Parade NYC website, rebuilt using React.
Live site: [https://persianparade.com/]
---
## Technologies
- JavaScript (majority of codebase)
- CSS
- HTML
- Dockerfile
---
## Prerequisites
Before getting started, ensure the following are installed:
- **Node.js** (version 14 or newer recommended)
- **npm**
---
## Getting Started
1. Clone the repository
```
git clone https://github.com/ColbyHman/persian-parade-website.git
cd persian-parade-website
```

2. Install dependencies
```
npm install
```
3. Start the development server
```
npm run start
```
